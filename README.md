# How Do You Feel?

An interactive static web application that maps user-reported symptoms to relevant acupuncture points, meridians, and body regions.

This project is designed as a symptom-to-point exploration tool. Users can describe what they are feeling, combine multiple complaints in a weighted query, and browse the resulting acupuncture points through a modern visual interface with body mapping and meridian context.

## Highlights

- Symptom-based search for complaints such as cough, insomnia, anxiety, gastric discomfort, and more
- Weighted multi-symptom queries such as `maag^2 + mual + cemas`
- Meridian filtering and quick exploration
- Interactive body map with `surface`, `organ`, and `meridian` layers
- Point detail view with related points
- Shareable search state through the URL
